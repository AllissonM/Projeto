Nosso Projeto

Nosso projeto tem como objetivo, demonstrar e informar curiosidades sobre o tema deep weeb, pouco esclarecido na internet. Para isso, desenvolvemos um site dinamico com design e layout tematicos e simples para que o usuario fique satisfeito com o conteudo. O site foi programado no visual code, usando ferramentas de html, css e javascript. No site, o conteudo está separado em varios topicos, tais como relatos, sobre, inicio e historia, cada um com propositos diferentes de informações.
